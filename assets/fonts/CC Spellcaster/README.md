# Spellcaster Font

This directory should contain the CCSpellcaster Regular.ttf font file.

The font is referenced in the CSS as:
```css
@font-face {
    font-family: 'Spellcaster';
    src: url('../assets/fonts/CC Spellcaster/CCSpellcaster Regular.ttf') format('truetype');
    font-weight: normal;
    font-style: normal;
}
```

If the font file is not available, the browser will fall back to serif fonts.
